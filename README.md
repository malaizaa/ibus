ibus - Extend Symfony 3 router with ability to dynamically load routes from database.
====

## Tech stack: ##
* Symfony 3.2.X
* Php 7.0.X
* PHPUnit 6.0.7

## Before start: ##
```
composer install
```

## Access route manage area: ##
`/manage/route/`

## For clearing route cache use this console command: ##
```
php bin/console app:clear-cached-routes
```

## Test run: ##
`phpunit`
